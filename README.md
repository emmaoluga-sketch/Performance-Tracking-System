# Performance Tracking System

> This repository demonstrates a structured performance management system designed for consistency, fairness, and data-driven insights.

## Problem
- Reviews conducted inconsistently
- Manager rating bias (calibration drift)
- No long-term performance visibility

## Solution
- Quarterly performance cycles
- Standardized evaluation rubrics
- Automated reminders
- Centralized reporting dashboard

## Tools
- Lattice
- Google Sheets
- Looker Studio
- Zapier

## Outcomes
- 96% on-time review completion
- +38% improvement in rating consistency
- Reduced cycle time by 2 weeks
